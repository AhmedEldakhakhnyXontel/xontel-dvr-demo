User Module Error Code Definitions

错误码的定义应尽量兼容HTTPstatus码，可以根据具体接口的需要进行扩展。

Error Code HTTP Status Code Description
200 HTTP_OK Processed successfully
400 HTTP_BAD_REQUEST Invalid request, missing parameters
401 HTTP_UNAUTHORIZED This user logged in elsewhere; current session invalid
403 HTTP_FORBIDDEN Illegal request, e.g., during factory reset password, the requesting user has not logged in on this phone; delete sharing
404 HTTP_NOT_FOUND Request not found; usually means the server has no handler for the client's request. Example: (1) Unsupported function message number in the request
405 HTTP_NOT_ALLOWED Request not allowed; the client lacks sufficient permissions, so the server will not execute the requested message content; e.g., (1) Client did not log in before requesting this message
406 HTTP_NOT_ACCEPTABLE Request not accepted; generally indicates the server did not accept the client's request; e.g., (1) Username and/or password incorrect during login
409 HTTP_CONFLICT Request conflict: already exists in database; e.g., user already registered, device ID already added
426 HTTP_UPGRADE_REQUIRED Client needs to upgrade
429 HTTP_TOO_MANY_REQUESTS Too many requests; e.g., feedback allowed only once per hour
451 HTTP_ILLEGAL Illegal access; indicates the client sent an illegal request; e.g., (1) Client session ID incorrect
500 HTTP_INTERNAL_SERVER_ERROR Server error; e.g., database operation error
501 HTTP_METHOD_NOT_IMPLEMENTED Specified method not implemented
502 HTTP_BAD_GATEWAY Invalid gateway
503 HTTP_SERVICE_UNAVAILABLE Service unavailable
504 HTTP_GATEWAY_TIMEOUT Gateway timeout
505 HTTP_VERSION_NOT_SUPPORTED This version is not supported
507 HTTP_INSUFFICIENT_STORAGE Invalid storage or insufficient storage space
508 HTTP_NO_ACTIVA_USER Account not activated
509 HTTP_DISABLE_USER Account disabled
510 Exceeds maximum limit
511 Incorrect email address
512 Login password incorrect
513 Insufficient permissions
514 Verification code error

Get Face Push

    使用CallCustomFuncExExHaveConnect 使用json 传递数据get，command:65791
    请求：
    {
        "Name":"SmartFace.Push"
    }
    返回：
    {
        "SmartFace.Push": {
            "PushEnable": 0,
            "BPushEnable": 0,
            "WPushEnable": 0,
            "NWPushEnable": 0
        },
        "Name": "SmartFace.Push",
        "Ret": 100
    }
    PushEnable：消息push总开关使能，如果APP不setdevice该parameters，忽略；
    BPushEnable：黑名单消息push，0关闭，1打开；
    WPushEnable：白名单消息push，0关闭，1打开；
    NWPushEnable：非白名单消息push，0关闭，1打开
